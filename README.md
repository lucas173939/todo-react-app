# Lista de Tarefas React

Aplicação desenvolvida com React para gerenciamento de tarefas, com persistência em `localStorage`.

## Funcionalidades
- Adicionar tarefas
- Marcar como concluída
- Remover tarefa
- Filtrar por todas, pendentes e concluídas
- Limpar tarefas concluídas
- Salvar dados no navegador

## Tecnologias
- React
- JavaScript
- CSS
- Vite

## Como executar
```bash
npm install
npm run dev
```

## Build de produção
```bash
npm run build
```
