# Lista de Tarefas - React

Aplicação desenvolvida em React para gerenciamento de tarefas do dia a dia.

O sistema permite adicionar, concluir e remover tarefas, com armazenamento local no navegador.

## Funcionalidades
- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Salvamento automático no navegador (localStorage)

## Tecnologias utilizadas
- React
- JavaScript
- HTML
- CSS

## Objetivo do projeto
Praticar conceitos fundamentais do React, como componentes, estado (useState) e manipulação de eventos.

## Como executar o projeto
1. Baixar o projeto
2. Instalar dependências
3. Executar o projeto

npm install
npm run dev
